### Messenger web app using 
* React
* Redux
* Express
* Typescript
* Sass
* Jest
* ESLint / StyleLint
* Webpack
* Yandex Cloud
* Git
* Postgres
* Nginx
* Docker

## Building and running
⚡ Installation
```
npm install
```
🐣 Running
```
npm start
```
Your app should now be running on http://localhost:3000/

📕 Eslint
```
npm run eslint
```

📗 Stylelint
```
npm run stylelint
```

🐙 Auto-tests
```
npm run test
```
